## 核心成分逐一看

### 银杏叶（Ginkgo Biloba）——"生命之树"

银杏叶是 iGenics 的核心成分。耶鲁大学的研究提出，银杏因其抗氧化和抗炎特性，可能有助于改善青光眼患者的视野指数，研究者表示它"可能预防或减缓视网膜神经节细胞死亡"。

需要注意的是：银杏与华法林、阿司匹林等抗凝药有相互作用，服用抗凝药的人在使用前务必咨询医生。

### 越橘（Bilberry）——WWII 飞行员的眼睛秘密

说来有趣，二战期间英国皇家空军飞行员执行夜间任务前会吃越橘酱，据说是为了提升暗视力。这个故事听起来像传说，但现代研究确实发现越橘中的花青素具有保护视网膜细胞的作用。

iGenics 在每份中使用480mg，是市面多数产品的4-6倍。

### 藏红花（Saffron）

作为世界上最贵的香料，藏红花含有强效抗氧化剂，多项临床试验显示其对与年龄相关的黄斑变性（AMD）有积极影响。iGenics 每份含20mg。

### 叶黄素（20mg）和玉米黄质（10mg）

这两种成分是眼睛中天然存在的唯一两种抗氧化剂，专门守护视网膜黄斑区域。发表在《美国医学会杂志》的研究显示，补充叶黄素和玉米黄质与降低 AMD 风险有显著关联。随着年龄增长，身体自身产量会下降，通过补充剂补充是目前主流眼科营养建议。

### 姜黄素（Turmeric，350mg）

除了担任"零填充剂"的制造助流角色，姜黄素本身有强大的抗炎效果。伦敦大学学院和帝国理工学院的研究发现，姜黄素中的活性成分对预防青光眼初期的视网膜神经节细胞损失有神经保护作用。

### 黑胡椒提取物（BioPerine，20mg）

BioPerine 是黑胡椒标准化提取物，能显著提升其他成分的生物利用率，让营养素更快进入血液发挥作用。
